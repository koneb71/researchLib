# research lib
This app will be submitted to MSU-IIT: College of Computer Studies department

### Setup

Prerequisite
 - Python 3.7 or later

Install Requirements
```pip install -r requirements.txt```

Setup Database
```python manage.py migrate```

Run
```python manage.py runserver```

### Features:
- ~~Google OAUTH2~~
- ~~Post Research Paper~~
- WYSIWYG editor
- PDF viewer
- Full Text Search
